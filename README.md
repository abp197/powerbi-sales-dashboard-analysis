
# 📊 Power BI Dashboard Analysis

## 📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze and visualize business data effectively. The dashboard provides insights through various charts, cards, and filters, helping users understand trends and key performance indicators.

---

## 🚀 Features

- Interactive Dashboard
- KPI Cards
- Column Charts
- Bar Charts
- Pie Charts
- Donut Charts
- Slicers for Dynamic Filtering
- Data Visualization and Analysis
- User-Friendly Interface

---

- 📊 Column Chart

1. Clone the repository:

```bash

```

2. Open `PROJECT_FRIST.pbix` using **Microsoft Power BI Desktop**.

3. Refresh the data if required.

4. Explore the dashboard and interact with filters.
